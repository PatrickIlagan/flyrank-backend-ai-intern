# flyrank-backend-ai-intern

Repository for the FlyRank AI Backend Engineering Intern experience (3 months).

## Structure

- `docs/` — reference documents and context for the internship (proof statements, briefs, etc.)
- `Week 1/` through `Week 10/` — weekly folders for goals, tasks, and notes as the internship progresses

Each weekly folder has its own `README.md` tracking that week's goals, tasks, and notes.
